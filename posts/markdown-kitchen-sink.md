# Markdown Kitchen Sink

The blog intentionally keeps posts as plain Markdown files, close to a small Jekyll site.

## Checklist

- Markdown images use normal `![alt](url)` syntax.
- Tables use md4c's GitHub-style table extension.
- Audio and video can be attached under `/assets/media`.
- CSS is generated dynamically from query parameters.

## Another Table

| --- | --- |
